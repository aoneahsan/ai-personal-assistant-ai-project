Product Adoption Widget - CDN Example

This example shows how to use the Product Adoption Widget via CDN. Simply include the script tag and initialize the widget.

Installation

Add this script tag to your HTML:

<script src="../dist/product-adoption-widget.min.js"></script>

Key Features

  • Zero dependencies
  • Lightweight (~50KB minified)
  • Works on any website
  • Mobile responsive

Analytics Integration

The widget automatically tracks user engagement and integrates with your existing analytics tools.
